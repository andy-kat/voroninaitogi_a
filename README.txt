Валютный конвертер с историей (Python + Tkinter)
Простая программа для конвертации валют с графическим интерфейсом, использующая бесплатный API ExchangeRate-API. История операций сохраняется в файл history.json.

Возможности программы
Конвертация валют по актуальному курсу.
Не требует регистрации или API-ключа.
История операций сохраняется между запусками.
В интерфейсе отображаются последние операции.
Атрибуция источника курсов (по требованию API).
Требования
Python 3.8+
Библиотека requests

Запустите программу:
В появившемся окне введите сумму в USD, выберите валюту и нажмите «Конвертировать».
История операций будет отображаться в нижней части окна и сохраняться в файл history.json.
Примеры использования

1. Конвертация 100 USD в EUR

Введите 100 в поле «Сумма в USD».
Выберите EUR из списка валют.
Нажмите «Конвертировать».
Результат появится под кнопкой, а операция будет добавлена в историю.

2. Просмотр истории
Все операции сохраняются в файле history.json.
В интерфейсе отображаются последние 100 операций с датой, суммой, валютой и курсом.


Программа использует бесплатный тариф ExchangeRate-API (Open Access), не требует ключа.
История хранится локально в формате JSON.
Для работы необходим доступ к интернету (для загрузки курсов).
Атрибуция
По условиям использования бесплатного тарифа API, программа содержит ссылку на источник курсов. Клик по ссылке скопирует адрес в буфер обмена.
